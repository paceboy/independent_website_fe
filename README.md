# independent_website_fe
独立开发者网站模版

1、包含功能：首页，博客，开发者工具，关于，隐私政策，服务条款。并接入了google analystics，google adsense，google 索引;

2、基于next.js框架;

3、如果使用这个模版，需要将域名更新为自己的域名，google相关的id替换为自己的。
